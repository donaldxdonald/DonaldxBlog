### npm安装

> npm install --save-dev webpack-dev-server@2.9.1

### webpack.config.js

```javascript
devServer: {
    contentBase: './dist',
    inline: true
}
```

### package.json

```json
scripts:{
    "dev": "webpack-dev-server --open"
}
```
